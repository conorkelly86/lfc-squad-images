# lfc-squad-images
A repo for Liverpool player squad images in PNG format. 
Includes json file for data on each player. 
A repo to store Liverpool Squad player images for use in other projects.
```mermaid
  graph TD;
      players.json-->image;
      players.json-->name;
      players.json-->position;
      image-->card;
      name-->card;
      position-->card
```
